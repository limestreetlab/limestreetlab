  • A Common-Sense Guide to Data Structures and Algorithms
  • Jay Wengrow
  • A short and for-dummy style book introducing basic concepts of algorithms like recursion and data structures, so reader should walk away knowing what it means for an algorithm to be faster than another and how one data structure is more suitable in an application than another. Everything is gently explained in layman's terms. He introduces algorithmic complexity as an asymptotic concept so is useful for comparing algorithms of different complexity like O(log2N) vs O(N2) but ones of identical complexity can actually substantially differ by a dropped constant factor such as O(3N) vs O(N). He then discusses, again gently, data structures from hash maps and stacks/queues to binary trees and graphs, all with simple examples.
